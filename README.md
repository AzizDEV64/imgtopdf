# 📄 Lab Photo → PDF

Fotoğraflarınızı hızlıca A4 boyutunda PDF'e dönüştürün. Tüm işlemler tarayıcınızda gerçekleşir — dosyalarınız hiçbir sunucuya yüklenmez.

🔗 **[Canlı Demo](https://azizdev64.github.io/imgtopdf)**

---

## ✨ Özellikler

- **Sürükle & Bırak** — Fotoğrafları doğrudan sayfaya sürükleyin veya tıklayarak seçin
- **Sıra Değiştirme** — Galeriden fotoğrafları sürükleyerek sıralayın
- **PDF Önizleme** — İndirmeden önce PDF'in nasıl görüneceğini kontrol edin
- **Sayfa Numarası** — Her sayfanın altında otomatik sayfa numarası
- **Açık / Koyu Tema** — Tek tıkla tema değişimi, tercih kaydedilir
- **Mobil Uyumlu** — Telefon ve tablette tam responsive tasarım
- **Gizlilik** — %100 istemci taraflı, sunucuya veri gönderilmez

## 📸 Desteklenen Formatlar

| Format | Uzantı |
|--------|--------|
| JPEG   | `.jpg`, `.jpeg` |
| PNG    | `.png` |
| WebP   | `.webp` |

> WebP dosyaları otomatik olarak JPEG'e dönüştürülür (tüm tarayıcılarla uyumluluk için).

## 🚀 Kullanım

1. Fotoğraflarınızı sürükleyip bırakın veya tıklayarak seçin
2. İsterseniz sıralarını değiştirin
3. Dosya adı girin (opsiyonel)
4. **PDF Oluştur** butonuna tıklayın
5. Önizleyin ve indirin

## 🛠️ Teknolojiler

- **HTML5 / CSS3 / JavaScript** — Framework yok, saf web teknolojileri
- **[jsPDF](https://github.com/parallax/jsPDF)** — PDF oluşturma
- **CSS Variables** — Tema sistemi
- **Canvas API** — Resim formatı dönüşümü
